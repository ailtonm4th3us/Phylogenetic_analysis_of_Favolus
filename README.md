# README
This repository was created with the aim of sharing scripts used in modeling and editing maps in Rstudio. It contains two scripts: Procedures_modeling_in_ENTML; and Median_raster_script. 
Both scripts are free to use in your own analyses. 

Article: “Phylogenetic analyses of *Favolus* Fr. (Polyporaceae, Basidiomycota) reveal their evolutionary and biogeographic history”<br/>
Authors: Avelino de Oliveira, A.M.; Alvarenga, R.L.M; & Gibertoni, T. B.
